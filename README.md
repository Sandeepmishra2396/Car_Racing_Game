# 🚗 Ultimate Car Racing Game

A car racing game built using Python Turtle Graphics.

## Features
- Enemy cars
- Score & High Score
- Collision Detection
- Keyboard Controls
- City-Themed Environment
- Game Over & Restart System

## Controls
- Left Arrow → Move Left
- Right Arrow → Move Right

## Technologies Used
- Python
- Turtle Graphics
